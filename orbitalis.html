The Orbitalis Project

2015

About

The Orbitalis Project is a web project with an artistic purpose—a collaboration with Hugo Baeta and Gianluca Martini. In the project, we used satellites' orbits information to create ornaments using a 3d printer. My role was predominantly in HTML, CSS, visual and concept design.

In the website, you can select a satellite and interact with various options to draw an orbit on your style. Also you can click the CAMERA button to change your perspective to the satellite's.
